# Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on the scraped books dataset to explore the data, identify patterns, and summarize key insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Files
- books.csv
- EDA analysis Task 2.py

## Analysis Performed
- Data Cleaning
- Summary Statistics
- Data Visualization

## Output
The analysis provides useful insights from the books dataset through statistics and visualizations.
